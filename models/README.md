# models/ — not in git

Model artefacts that are produced (not downloaded) by this project: fitted axiom-based
surrogates (RQ5), calibration models, cascade thresholds (RQ6). Name them
`<experiment>__<short-description>/` and store the producing config alongside.

Downloaded LLM weights do **not** live here — they go to the Hugging Face cache under
`data/cache/huggingface/` automatically.
